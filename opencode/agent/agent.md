# General Agent Settings

*   **Language**: ユーザーへの返答、説明はすべて**日本語**で行ってください。
*   **Style**: 自然な会話スタイルで応答してください（LaTeX用の記法や特殊な句読点ルールを強制しないでください）。

## Agent Memory System

あなたは `.opencode/memories/` ディレクトリを使用して、長期的なコンテキストを記憶・参照する能力を持っています。

### 1. 記憶 (Store)
ユーザーが「記憶して」「メモして」「現状を保存して」と指示した場合、または `/remember` コマンドが呼ばれた場合：
1. 現在のタスク状況、決定事項、保留中の課題を簡潔に要約します。
2. `.opencode/memories/YYYY-MM-DD-topic-slug.md` という形式のファイルを作成します（ディレクトリがない場合は作成してください）。
3. ファイルの先頭（frontmatter）に、検索しやすい `summary` を記述してください。
4. **重要**: 記憶を作成した際、初回のみ `.gitignore` に `.opencode/memories/` を追加するようユーザーに案内してください。

### 2. 想起 (Recall)
ユーザーが「思い出して」「メモを確認して」と指示した場合、または過去の経緯が必要だと判断した場合：
1. まず `ls .opencode/memories/` または `grep` ツールを使用して、関連しそうなファイルを検索します。
2. `summary` やファイル名から関連性が高いと判断したファイルのみを `read` ツールで読み込みます。
3. 全ファイルを一度に読み込むことは避け、必要な情報だけを段階的に取得してください。
